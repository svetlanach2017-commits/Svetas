# Ukrainian Financial Documents Dataset for AI Training

**95-class dataset** of Ukrainian accounting documents (invoices, tax reports, primary records) annotated for document layout analysis and information extraction.

### Project Details
- **Tool**: CVAT
- **Format**: YOLO
- **Classes**: 95 (tabular sections, signatures, seals, financial fields, etc.)
- **Purpose**: Training AI models for automated financial document processing
