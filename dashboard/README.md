# Business Intelligence Dashboards Dataset

**Real annotated Power BI and Excel dashboards** — high-quality dataset for training AI models on dashboard understanding, data visualization and KPI interpretation.

### Purpose
This dataset is designed to train AI models to:
- Understand complex business dashboard layouts
- Extract and interpret KPI cards, charts, trends and comparative elements
- Automatically parse business metrics and visualizations used in real companies

### Dataset Details
- **Number of classes**: 54
- **Format**: YOLO
- **Annotation tool**: CVAT
- **Dashboard types**: Power BI + Excel
- **Content**: Sales performance, revenue, product analysis, monthly/annual metrics, filters and slicers

### Key Classes
- Chart Area, Plot Area, Chart Title, Axis Titles
- KPI Cards, Data Labels, Metrics (revenue, sales, percentage, growth/fall)
- Bar Charts, Line Charts, Donut Charts
- Filters, Slicers, Control Panels, Buttons
- Legends, Totals, Section Dividers

### Folder Structure
- `/dataset/` — original dashboard images and annotation files
- `/Examples/` — visual examples of annotations with bounding boxes

### Value for AI Training
Helps AI models learn to read and understand real business intelligence dashboards used for decision-making, performance monitoring and executive reporting.
