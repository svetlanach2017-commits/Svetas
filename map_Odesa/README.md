# Odesa Region Geospatial Map Dataset

**Annotated administrative map of Odesa Oblast** — dataset for training computer vision models in geospatial analysis and district recognition.

### Purpose
This dataset is designed to train AI models to:
- Recognize administrative districts on maps
- Perform geospatial segmentation
- Support urban planning, logistics and regional analysis tasks

### Dataset Details
- **Number of classes**: 7
- **Format**: YOLO
- **Annotation tool**: CVAT
- **Image**: High-resolution map of Odesa Region (Ukraine)
- **Classes**: Administrative districts of Odesa Oblast

### Classes
- Odessky district
- Izmail district
- Bolgradsky district
- Bilgorod-Dnistrovsky district
- Podilsky district
- Berezivsky district
- Rozdilny district

### Folder Structure
- `/dataset/` — original map image and annotation files
- `/Examples/` — visual examples of annotations with bounding boxes

### Value for AI Training
Helps models learn to identify Ukrainian administrative divisions on maps, useful for geospatial intelligence, logistics optimization and regional data analysis.
